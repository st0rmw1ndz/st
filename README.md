# st

Personal build of the simple terminal implementation for X

## Installation

```
# make clean install
```

## Patches

- [scrollback ringbuffer](https://st.suckless.org/patches/scrollback/) - allows you to scroll back
- [alpha](https://st.suckless.org/patches/alpha/) - allows users to change the opacity of the background
- [bold is not bright](https://st.suckless.org/patches/bold-is-not-bright/) - makes bold text rendered simply as bold
